# Dweeaboo
Hosting a website for weaboos and dweebs on an EC2 instance using FastAPI. Populating a MySQL database with https://jikan.moe/ API through AirFlow. 
# Technologies used
* Docker
* Airflow (Scheduler)
* FastAPI
* MySQL
* Pandas / SQLAlchemy (Data Manipulation)
* Requests (API's)
* EC2 & S3 (AWS)
